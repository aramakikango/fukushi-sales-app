使い方

- ファイル `transcription_samples.txt` に複数の iPhone 文字起こし風サンプルを用意しました。
- テスト手順（手動）:
  1. `reports.html` をブラウザで開きます（Web アプリまたは Apps Script のエディタでプレビュー）。
  2. 「iPhoneの文字起こしから自動入力」パネルのテキストエリアに、`scripts/transcription_samples.txt` の任意のサンプルをコピーして貼り付けます。
  3. 「解析してプレビュー」を押して解析結果を確認します。
  4. 必要に応じてプレビュー結果を編集し、「フォームに反映」を押してフォームに値を流し込みます。

- 備考:
  - サンプルはラベルあり/なし、複数担当、メールが名前欄に入っている等のケースを含めています。
  - 更にテストしたいパターンがあれば、このファイルに追加してください。
